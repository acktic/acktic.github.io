# RSS-Browser

  - Cross Device Browser.


### Features

* Image Parsing
* XML Examples
* Random in Category
* Private Proxy Requests
* Inline Image Expansion
* Opposite Visual Scheme


### Dependencies

Browser Tested on
* Apple Safari
* Google Chrome
* Mozilla Firefox


### Menu Array

* Array Structure in rss.js

`{ id : "Courtesy", cat : "Category", des : "Description", uri : "Feed", ext : "URL" },`
